• If the Main door is at the North of North-West direction(ie. if the door is facing the North side), then it is considered reasonably good.

• If the Main door is at the West of North-West direction(ie. if the door is facing the West side), then it is considered a flaw(Vastu dosha).

• The Main door should open inwards in a clockwise direction.

• Install a light outside the main door and keep it switched on during the evening hours.

• Install the ‘SACRED SYMBOL’ of Om or Swastik above the Entrance Door frame(install the symbols inside and outside).

• Above the Main Door, Install a pure Silver Coin depicting Religious Symbols of Om/Swastik/Om-Swastik Trishul/Lord Ganeshaji/Ek-Onkar/Navkar.

• Since the Main Door is not well positioned, Install the ‘POWER GRID STICK’ above the Main Door Frame(at the inner side/inside the property).\no Option 1: Screw/fix 2 'white metal hooks' on the wall(above the main door frame)(keep about 1.5 feet distance between the 2 'hooks'). Hang/place the ‘POWER GRID STICK’ between the 2 'hooks'.\no Option 2: Embed the ‘POWER GRID STICK’ within the wall(above the main door frame). Create a horizontal hole in the wall(hole should be measuring 25 inches long x 1.5 inches deep). Place the ‘POWER GRID STICK’ in it. Thereafter re-plaster the wall.
