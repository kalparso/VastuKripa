• Main door located at the West direction is considered reasonably acceptable.

• Above the Main Door frame, Install a picture of Lord Hanumanji(inorder to gain Divine protection from the negativities).

• Install the ‘SACRED SYMBOL’ of Om or Swastik above the Entrance Door frame(install the symbols inside and outside).

• If you have installed a moorti of Lord Ganesha ji above the Main Door from outside, then install a moorti of Lord Ganesha ji above the Main Door from inside the home also(ie. the moorti/picture of Lord Ganesha ji should be installed back to back).

• Since the Main Door is not well positioned, Install the ‘POWER GRID STICK’ above the Main Door Frame (at the inner side/inside the property).\no Option 1: Screw/fix 2 'white metal hooks' on the wall(above the main door frame)(keep about 1.5 feet distance between the 2 'hooks'). Hang/place the ‘POWER GRID STICK’ between the 2 'hooks'.\no Option 2: Embed the ‘POWER GRID STICK’ within the wall(above the main door frame). Create a horizontal hole in the wall(hole should be measuring 25 inches long x 1.5 inches deep). Place the ‘POWER GRID STICK’ in it. Thereafter re-plaster the wall.

• General attributes of West zone(Paschim disha):
West Ruled by: Lord Varuna(The God of Rain and Water)
West Ruled by: Saturn Planet(Shani)
Color: Black, Blue.
Mantra to be recited: 'Om Pram Prim Proom Sah Shanischarey Namah'.
Gemstone: Blue Sapphire(Neelam/Neelima).
